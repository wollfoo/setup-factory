# Điều phối Agents

Hãy điều phối các agents để thực hiện task này.

## Quy trình

1. **Phân tích task**: Xác định yêu cầu và phạm vi công việc
2. **Phân công agents**: Chia nhỏ task cho từng agent chuyên biệt
3. **Điều phối thực thi**: Quản lý luồng làm việc giữa các agents
4. **Tổng hợp kết quả**: Tích hợp output từ các agents
5. **Xác minh**: Kiểm tra chất lượng và tính đầy đủ

## Nguyên tắc

- Phân tách rõ ràng trách nhiệm
- Truyền thông context đầy đủ giữa agents
- Xử lý lỗi và fallback
- Theo dõi tiến độ và báo cáo
